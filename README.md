# Lanka Smart Mart (Beta)

Android app project for Lanka Smart Mart — local shopping and delivery.

## Quick start

- Prerequisites: Android Studio or JDK + Gradle
- Open the project in Android Studio or run from the command line with the Gradle wrapper:

```bash
./gradlew assembleDebug
./gradlew installDebug
```

- Use the `app/google-services.json` if using Firebase features.

## Notes

- This repo contains build artifacts in `app/build/` — avoid committing generated files.
- See `.gitignore` for excluded files.
